== Description ==

NertWorks Social Share Tools make it easy for you to place social share, add, follow and like buttons through out your website.  You can pick specific pages to make it show up with the [get_social_code] shortcode.  Or you can use the options within the plugin settings to make it show where you'd like.


== Installation ==
1. Upload `nertworks-social-share-tools` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize your buttons the way you want.  
4. Share your site til the cows come home.
